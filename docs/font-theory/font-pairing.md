# Font Pairing

## Recommended Font Combinations
1. **Serif + Sans Serif**:
   - Heading: `Merriweather` (Serif)
   - Body: `Open Sans` (Sans Serif)

2. **Modern Sans Serif**:
   - Heading: `Poppins` (Sans Serif)
   - Body: `Lato` (Sans Serif)

### Example CSS:
```css
h1, h2, h3 {
  font-family: 'Merriweather', serif;
}

body {
  font-family: 'Open Sans', sans-serif;
}
